== Description ==

Embed [NewsUp](http://www.newsup.com) quizzes on your wordpress site.
Allows the use of a special short code `[newsup]` for embedding NewsUp quizzes.

= Current Features =

* Embed quizzes as any type of user instead of just Editors and Admins

= Example: =
`[newsup quiz="48370292-540c-4483-b700-c59158edce13"]`

== Installation ==

1. Copy the plugin files to the `/wp-content/plugins/newsup` directory.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Plugin Name screen to configure the plugin
